# Desta Portfolio — Quick Deploy

## Netlify (super easy)
1. Go to https://app.netlify.com/drop
2. Drag-and-drop the `desta-portfolio` folder (or the ZIP) here.
3. Netlify will give you a live URL like `https://your-site-name.netlify.app`.
4. Optional: set a custom site name in Site Settings.

## GitHub Pages
1. Create a new GitHub repo (public).
2. Upload the contents of `desta-portfolio` (index.html + images).
3. Go to Settings → Pages → Build and Deployment → Deploy from branch → `main` / root.
4. Open the URL shown by GitHub Pages.
